# Analiza i przetwarzanie dźwięku - Projekt #1

C# / [NAudio](https://github.com/naudio/NAudio)

## Features

- Wczytywanie plików wav
- Wyświetlanie przebiegu czasowego pliku audio
- Wyświetlanie parametrów (wartości, wykresy)
- Detekcja ciszy
